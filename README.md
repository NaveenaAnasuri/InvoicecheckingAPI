# Invoice-checking-AP
This project is designed for Invoice Validation &amp; Anomaly Detection API
##
Invoice Validation & Anomaly Detection API
Problem:
Finance teams manually validate invoices.
Dataset / Input:
Sample invoice CSV/Excel data (Kaggle)
Logic to Build:
Rule-based checks (amount mismatch, duplicate invoice)
Excel automation for preprocessing
API Contract:
Input: Invoice JSON
Output: Validation status + flags
Outcome:
An invoice-checking API ready for ERP integration.
